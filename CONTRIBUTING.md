# Contributing

This repository is maintained as a research artifact. Contributions should preserve reproducibility, clinical-safety framing, and the existing `src` package layout.

## Development

1. Create a local environment.
2. Install the package with `pip install -e .`.
3. Run `python -m pytest tests -q` before proposing changes.
4. Do not commit virtual environments, caches, coverage reports, generated temporary files, or large local experiment outputs.

## Scope

Keep changes focused on code correctness, documentation accuracy, reproducible experiments, and verified figures or results.
