# Security Policy

## Scope

This repository is a research software artifact. It is not a medical device and must not be used for clinical decision-making without independent validation and institutional approval.

## Reporting

Report security or data-safety concerns privately to:

- Chatchai Tritham: `chatchait66@nu.ac.th`
- Chakkrit Snae Namahoot: `chakkrits@nu.ac.th`

Do not publish sensitive clinical data, private credentials, or exploitable details in public issues.

## Data Safety

- Do not commit patient-identifiable data.
- Do not commit secrets, API keys, access tokens, private certificates, or local environment files.
- Keep generated caches, virtual environments, and coverage artifacts out of version control.
